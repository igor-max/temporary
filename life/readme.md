

读书的看法

1. 一本书的主线自由一个，其他横向纵向只是对他的补充（参考金字塔原理），补充的观点可能会延申其他的一些观点（我们抓住主要观点就行，因为其他观点也会在另一本书作为主观点去论证）
2. 那么对主观点的论真，我们可以同意也可以不同意（这跟我们的生活层次有关）重要的是在生活中怎么应用你认为好的观点（知行和一）那么这本书对于自己来说就是有收货的

关键字

**主线（主题）**   **在生活中应用**
**不要被支线绕住（这样看书就会累）**
